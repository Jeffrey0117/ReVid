.revid format

A video as a pointer把影片變成一個指標

A .revid file is a tiny JSON that points to your uploaded video — no pixels, no gigabytes. Double-click it and ReVid streams it back like it never left. Part of a shared format the whole trio (RePic / ReVid / MemoryGuy) understands.一個 .revid 檔只是個小 JSON,指向你上傳的影片 — 沒有畫面資料、不佔 GB。雙擊它,ReVid 就把影片串回來,像沒離開過。這是三棟(RePic / ReVid / MemoryGuy)共用的格式。
